# 🔢 Huge Integer (C++ BigInt)

A C++ implementation of a "Huge Integer" class, designed to handle and perform arithmetic on numbers that far exceed the limits of standard data types like `long long`. This project demonstrates a deep understanding of string-to-numeric conversion and manual arithmetic logic.

## ✨ Key Features

* **Infinite Precision:** Handle numbers with hundreds or even thousands of digits.
* **Basic Arithmetic:** Supports addition, subtraction, and comparison of large integers.
* **OOP Design:** Entirely built using a `HugeInteger` class for clean, reusable code.
* **String-Based Logic:** Utilizes string manipulation to store digits and simulate "carry/borrow" math.

## 🧠 How it Works

Standard C++ integers are capped at 64 bits. This project overcomes that by:
1. Storing the number as a **string** or **vector of integers**.
2. Implementing manual addition and subtraction from right to left (like we do on paper).
3. Handling negative results and sign management for complex calculations.

## 🛠️ Technologies Used

* **Language:** C++
* **Logic:** Custom arithmetic algorithms, String manipulation.
* **Environment:** Microsoft Visual Studio / Any C++ Compiler.
